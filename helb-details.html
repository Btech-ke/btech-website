HELB Loan Application Details

To apply for the HELB loan, ensure you meet the following requirements:

Requirements for First-Time Applications:

  • KCSE Index Number
  • KCPE Index Number
  • Birth Certificate Number
  • Parents’ ID copies, death certificate (if one parent is deceased), and guardian’s ID (if orphaned)
  • Student's ID photocopy
  • KRA pin certificate copy for the student
  • KRA pin certificate for the parent(s), if applicable
  • Admission letter and fee structure
  • One passport-sized color photo
  • The student must have an email address and phone number registered with their ID
  • Two guarantors' ID copies (parents can act as guarantors)
  • A banker’s card (Equity, Coop, KCB, or ABSA banks; SACCO accounts are not accepted)

How to Apply as a Second and Subsequent Applicant:

If you were awarded a scholarship and/or loan in the previous academic year, you can apply for a second or subsequent loan by:

  • Dialing *642# on your mobile phone
  • Downloading the HELB Mobile App from the Google Play Store
  • Using the MPESA Mini App
